# REST-API "Менеджер заметок"

## Описание

Цель проекта: закрепить на практике теоретические знания по стеку Java/Spring.

## Стек технологий:

* Java SE 25
* PostgreSQL
* Maven
* Spring Framework 7
* Spring Boot 4
* Spring Web
* Spring Data
* Hibernate ORM
* Hibernate Validation
* Hibernate Processor
* Liquibase
* JUnit
* Mockito
* Devtools
* Actuator и Admin
